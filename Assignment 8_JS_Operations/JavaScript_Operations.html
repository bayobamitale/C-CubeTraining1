JavaScript Operations

JavaScript can be used to declare variables to be operated upon by operators like +, -, *, %, =, e.t.c

Variables are containers to hold or store data. These variables are called operands when operated on by operators. For example, you can declare variables firstNumber and secondNumber and assign values to each using the let keyword and the assignment operator as follows: 

				let firstNumber = 20;
				let secondNumber = 30;
				let sum = 0;

Expression narrates the relationship between the operands and their operators. Example: 

				sum = firstNumber + secondNumber

Statement is a terminated expression. Example: 

				sum = firstNumber + secondNumber;

A block is a collection of statements to achieve a specific task. This collection of statements are usually enclosed in curly braces. Example of block that adds two numbers together and prints the result: 

				function Add()
				{
					let firstNumber = 20;
					let secondNumber = 30;
					let sum = 0;
					sum = firstNumber + secondNumber;
					window.alert(sum);
				}
