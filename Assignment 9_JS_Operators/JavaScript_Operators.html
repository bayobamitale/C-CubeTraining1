Understanding JavaScript Operators

JavaScript can be used to perform operations using operators like +, -, *, /, %, =, ==, !=, >, <, >=, <=, e.t.c to achieve user set goals. This work describes these operators with examples.

Addition

Addition operation can be performed on variables using the "+" symbol to calculate the sum of two or more numbers. For example, you can declare variables firstNumber and secondNumber and assign values to each using the let keyword and the assignment operator "=" and add the two variables together using the addition operator "+". The sum of the two numbers is then store in the variable sum. 

				let firstNumber = 20;
				let secondNumber = 30;
				let sum = firstNumber + secondNumber;

Subtraction

Subtraction operation can be performed on variables using the "-" symbol to calculate the difference of two or more numbers. For example, the difference of the two numbers is shown below: 

				let firstNumber = 20;
				let secondNumber = 5;
				let difference = firstNumber - secondNumber;

Multiplication

Multiplication operation can be performed on variables using the "*" symbol to calculate the product of two or more numbers. For example, the product of two numbers is shown below: 

				let firstNumber = 20;
				let secondNumber = 5;
				let product = firstNumber * secondNumber;

Division

Division operation can be performed on variables using the "/" symbol to calculate the division of numbers. For example, the division of two numbers is shown below: 

				let firstNumber = 20;
				let secondNumber = 5;
				let division = firstNumber / secondNumber;

Modulus

Modulus operation can be performed on variables using the "%" symbol to get the remainder of the division of numbers. For example, the remainder of the division of two numbers can be determined as shown below: 

				let firstNumber = 20;
				let secondNumber = 5;
				let remainder = firstNumber % secondNumber;

Equality

Equality operation can be performed on variables using the double equal sign "==" to test the equality of numbers. It returns a boolean "true" if numbers are equal, else it returns "false". For example, the following returns a false since the numbers are not equal: 

				let firstNumber = 20;
				let secondNumber = 5;
				let equal = firstNumber == secondNumber;

Not Equal

The "Not Equal" operation can be performed on variables using the sign "!=" to test the non equality of numbers. It returns a boolean "true" if numbers are not equal, else it returns "false". For example, the following returns a true since the numbers are not equal: 

				let firstNumber = 20;
				let secondNumber = 5;
				let equal = firstNumber != secondNumber;

Comparison Operators

The comparison operators consist of, among others, the equality and not equal (discussed above), greater than, less than, greater or equal to, less than or equal to operations (==, !=, >, <, >=, <=). It returns a boolean "true" if the condition is satisfied, else it returns "false". For example, the program below prints out true since 20 is greater than 5: 

				let firstNumber = 20;
				let secondNumber = 5;
				let compare = firstNumber > secondNumber;
				alert(compare);
